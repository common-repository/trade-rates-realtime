== Description ==

One simple, light and fast wordpress plugin-widget that shows the forex rates in realtime. The quotes comes in real time from forex server or you can choose your own one. The plugin have control panel, where you can set the title, fields for preferred forex quotes and the url of the trade server that you want to use.

== Installation ==

1. Unzip "forex-rates-wp-widget.zip" and upload the contained files to the "/wp-content/plugins/"  directory
2. Activate the plugin through the "Plugins" menu in WordPress

= Add this widget to sidebar =

1. Install "Forex rates - Realtime" through the WordPress admin menu of Appearance or Design and then widgets.

= Add this realtime rates monitor to a post or a page =

1. Place [wp_forex_rates] in the content to insert into a post.
